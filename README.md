# healthcare-analytics-dashboard
## 📌 Project Overview
This dashboard provides insights into healthcare utilization, patient demographics, and payer distribution.

## 🛠️ Tools Used
- Power BI
- Excel / SQL (if used)

## 📊 Features
- Encounters by payer analysis
- Top diagnosis categories
- Time trend of encounters
- Age distribution of patients

## 📈 Key Metrics
- Total Encounters: 21,669
- Unique Patients: 3,286
- Median ER Time: 60 mins
- Avg Length of Stay: 4.5 days
